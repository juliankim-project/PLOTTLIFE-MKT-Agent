# Product Marketing Context

*Last updated: 2026-04-16*

## Product Overview

| 항목 | 내용 |
|------|------|
| **One-liner** | 한국에서 가장 쉬운 단기임대 방 찾기 플랫폼 |
| **What it does** | 유학생, 출장자, 이사 중인 사람 등 단기 거주가 필요한 사람과 공실을 수익화하려는 호스트를 연결하는 단기임대 마켓플레이스. 1주~20주 단위로 풀옵션 가구 완비 방을 계약하고, 결제·입주·퇴실까지 플랫폼이 관리. |
| **Product category** | 단기임대 / 주거형 숙소 마켓플레이스 |
| **Product type** | 양면 마켓플레이스 (Two-sided marketplace) |
| **Business model** | 수수료 기반 — 성사된 계약 건당 커미션 |

## Key Differentiators

| 차별화 포인트 | 설명 | 경쟁사 대비 |
|---------------|------|-------------|
| **ARC 발급 지원** | 거주숙소제공확인서 발급 | 경쟁사 중 유일 |
| **인증 매물** | 플랫폼이 직접 검증한 방만 노출 | 직방/다방 사기 매물 문제 해결 |
| **즉시 계약** | 호스트 승인 대기 없이 바로 계약 | 에어비앤비 호스트 승인 대기 제거 |
| **최저가 보장** | 동일 매물 타 플랫폼 대비 가격 보장 | 가격 신뢰 확보 |
| **다국어 지원** | 한/영/중/베/일 5개 언어 | 직방/다방 한국어 전용 |
| **1주 단위 계약** | 에어비앤비(1박)와 월세(12개월+) 사이 | 빈 구간 독점 |

## Jobs to be Done

### 게스트
1. 한국에서 안전하고 검증된 단기 방을 빠르게 구하고 싶다
2. 복잡한 계약 절차 없이 온라인으로 바로 계약하고 싶다
3. 보증금 부담 없이 입주하고 싶다
4. ARC(외국인등록증) 발급에 필요한 거주지 서류를 구하고 싶다

### 호스트
1. 빈 방을 놀리지 않고 수익을 만들고 싶다
2. 계약·결제·입퇴실 관리를 직접 안 하고 싶다
3. 외국인 게스트 대응(언어, 서류)을 위임하고 싶다

## Objections & Responses

| Objection | Response |
|-----------|----------|
| "에어비앤비랑 뭐가 다른가요?" | 에어비앤비는 여행 숙소, 플라트라이프는 생활형 숙소. 1주~수개월 단위 계약, 풀옵션 가구, ARC 서류 발급까지 지원. |
| "직접 방 보지 않고 계약해도 되나요?" | 인증 매물은 플랫폼이 직접 검증. 사진·영상·상세 정보 제공. 입주 후 문제 시 고객지원팀 대응. |
| "보증금을 돌려받을 수 있나요?" | 보증금은 플라트라이프가 보관. 퇴실 후 확인 절차 거쳐 환불. 호스트에게 직접 맡기는 구조가 아님. |

### Anti-Persona
- **게스트:** 1박~3박 여행자 (호텔/에어비앤비가 더 적합), 전세 자금이 있는 장기 거주자
- **호스트:** 숙소 직접 관리·대면 소통을 선호하는 호스트, 1박 단위 에어비앤비 운영자
