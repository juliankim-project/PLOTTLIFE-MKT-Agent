# Brand & Language Guide

*Last updated: 2026-04-16*

## Brand Voice

| 항목 | 내용 |
|------|------|
| **Tone** | 친절하고 실용적, 신뢰감 있는 |
| **Style** | 간결하고 직관적. 복잡한 프로세스를 쉽게 설명. 외국인도 이해할 수 있는 명확한 표현. |
| **Personality** | 따뜻한(warm), 믿을 수 있는(trustworthy), 실용적인(practical), 학생 친화적인(student-friendly) |
| **Color palette** | 따뜻한 어스톤 (#74594b 브라운 기반, #f7f5f2 크림 배경) |

## Customer Language

### 고객이 문제를 설명하는 말 (게스트)
- "한국에서 방 구하는 게 너무 어려워요"
- "에어비앤비로 한 달 살면 200만원 넘게 나와요"
- "부동산에 전화했는데 외국인이라고 거절당했어요"
- "ARC 발급받으려면 거주지 확인서가 필요한데 구할 수가 없어요"

### 고객이 우리를 설명하는 말 (게스트)
- "외국인이 한국에서 방 구할 수 있는 사이트"
- "에어비앤비보다 싸고 생활하기 좋은 방"
- "ARC 서류도 해주는 단기임대"

## Words to Use / Avoid

| Use | Avoid | 이유 |
|-----|-------|------|
| 단기임대 | 숙소 예약 | "예약"은 여행 느낌 |
| 풀옵션, 가구 완비 | 숙박 | "숙박"은 호텔 느낌 |
| 즉시 입주 | 렌트 | "렌트"는 자동차 느낌 |
| 유학생, 한달살기 | 민박 | 비전문적 느낌 |
| 안전한, 검증된 | — | — |
| 간편 계약 | — | — |

## Glossary

| Term | Meaning |
|------|---------|
| 플라트라이프 (plott LIFE) | 서비스 브랜드명 |
| ARC | Alien Registration Card, 외국인등록증 |
| 거주숙소제공확인서 | ARC 발급에 필요한 서류, 호스트가 발급 |
| 인증 매물 | 플랫폼이 직접 검증한 방 |
| 즉시 계약 | 호스트 승인 대기 없이 바로 계약 가능 |
| 최저가 보장 | 동일 매물 타 플랫폼 대비 최저가 약속 |
| 풀옵션 | 가구·가전 완비 (세탁기, 냉장고, 에어컨, 침대 등) |

## Proof Points

### Value Themes

| Theme | Proof |
|-------|-------|
| 외국인 특화 | 5개 언어 지원 + ARC 서류 발급 — 경쟁사 중 유일 |
| 안전한 거래 | 인증 매물 + 보증금 플랫폼 보관 (호스트 직접 보관 아님) |
| 유연한 기간 | 1주 단위 계약 — 에어비앤비(1박)와 월세(12개월+) 사이 빈 구간 |
| 합리적 가격 | 최저가 보장 + 보증금 없는 방 옵션. 에어비앤비 월 200만원+ vs 플라트라이프 40~70만원 |
