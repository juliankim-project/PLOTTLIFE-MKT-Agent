12 · 조정·이터레이션 가이드

🛠 어떻게 조정하나요?

  1. 전역 색상·폰트·간격 조정 → shared.css 최상단 :root 변수 수정. 모든 페이지 즉시 반영.
  2. 컴포넌트 스타일 조정 → shared.css 해당 섹션 (예: /* ── Cards ── */) 안의 클래스 규칙 수정.
  3. 새 컴포넌트 추가 → shared.css 에 새 클래스 추가 → 이 페이지(kit.html)에도 예시 추가.
  4. 페이지별 예외 스타일 → 각 HTML 파일 <style> 블록에만 지역 추가.
  5. 새 페이지 만들 때 → A-seongsu.html 구조를 복제 → 본문 교체.
